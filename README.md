# Physical AI & Humanoid Robotics Textbook

Welcome to the Physical AI & Humanoid Robotics Textbook project. This textbook aims to provide comprehensive coverage of:

- Physical AI concepts
- ROS 2 basics
- Gazebo simulation
- Unity visualization
- NVIDIA Isaac Sim and Isaac ROS
- Vision-Language-Action systems
- Humanoid robot kinematics
- Conversational robotics using GPT models
